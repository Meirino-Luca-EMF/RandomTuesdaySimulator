## Coconut Guard
Si Coconut.jpg est supprimé, l'app affiche un écran d'erreur et refuse de démarrer.
Restaure le fichier et recharge -> tout refonctionne.

## Contrôles In-Game
```
W/A/S/D  - Déplacement
E        - Interagir avec les objets
Q        - Inventaire
ESC      - Pause
```
## Pour lancer
Ouvre src/app/Index.html dans un navigateur moderne.
